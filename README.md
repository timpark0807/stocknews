# Stock News
Welcome to the Reddit `!stocknews` bot repository

## Introduction 
To use this service, post a comment in the /r/investing subreddit with the format `!stocknews $TICKER`
### Examples 

`!stocknews $TSLA`

`!stocknews $AAPL`

`!stocknews $UBER`

## Technologies Used
- Python 
- PRAW
- Reddit API
- IEX Cloud API 
- AWS Lambda
- AWS Cloudwatch Events 
- REST API
